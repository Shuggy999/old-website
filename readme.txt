== Description ==

This is a free demonstration version of the Back to The 90s Retro Theme for WordPress. This makes your site look like a real blast from the past! However, under the hood runs a modern HTML5, responsive, SEO-friendly, easily customized, translation ready, crossbrowser compatible code.

Use "Appearance -> Customize" to control the details of your site. 

The theme has two columns by default. If you place widgets to the right hand panel a third column will appear. On mobile devices all the content is displayed on a single column.
